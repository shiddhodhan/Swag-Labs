## 📌 Overview

This project is a **Selenium automation framework** designed to test the SauceDemo web application. It automates key user journeys like login, product selection, cart operations, and checkout.

Built using **Java + Selenium + TestNG + Maven**, this framework follows best practices for scalability and maintainability.

---

## 🚀 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* DDD

## 📂 Project Structure

```
src
 ├── main/java
 │    ├── base        # Driver setup
 │    ├── pages       # Page Object classes
 │    └── utils       # Reusable methods
 │
 └── test/java
      ├── tests       # Test cases
      └── runners     # Test runners (if used)
```

## ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/shiddhodhan/Swag-Labs.git
```

### 2️⃣ Navigate to project

```
cd Swag-Labs
```

### 3️⃣ Run tests

```
mvn clean test
```
